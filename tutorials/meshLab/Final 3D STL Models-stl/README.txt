OVERVIEW
This folder contains geometries/images/segmentation/etc... created from the 
National Libraries of Medicine Visible Human Female and Male cryosections. The
work provided herein was created by the Center for Orthopaedic Biomechanics at the
University of Denver and is free to use under the Creative Commons Attribution 4.0 license.

You may use the files provided for any purpose following the license guidelines but we ask that
you cite the original papers in any publications/patents/abstracts/software/etc... that are derived
using any portion of this work. We ask that you cite the following two works that describe the 
data included and the processing work used to create the data.

Andreassen, T. E., Hume, D. R., Hamilton, L. D., Walker, K. E., Higinbotham, S. E., & Shelburne, K. B.
"Three-dimensional lower extremity musculoskeletal geometry of the Visible Human Female and Male."
Scientific Data, 2022.
doi: https://doi.org/10.1038/s41597-022-01905-2

Andreassen, T. E., Hume, D. R., Hamilton, L. D., Higinbotham, S. E. & Shelburne, K. B.
"An Automated Process for 2D and 3D Finite Element Overclosure and Gap Adjustment using Radial Basis Function Networks."
Arxiv, 1–13 2022.
doi: https://doi.org/10.48550/arXiv.2209.06948


The full dataset is located at the following doi:
https://doi.org/10.56902/COB.vh.2022.0

If you have any questions about this dataset please email Thor Andreassen at thor.andreassesn@du.edu
and Kevin Shelburne at kevin.shelburne@du.edu
